# Asset Management Register – ABC Financial Services Ltd

## Project Overview

This project demonstrates the creation of a comprehensive IT Asset Register for a fictional financial services company.

The register includes:

- Hardware Assets
- Software Assets
- Information Assets
- Service Assets
- Physical Locations
- Personnel

The project follows cybersecurity asset management principles by documenting ownership, classification, lifecycle, support information, recovery objectives, and backup locations.

## Skills Demonstrated

- IT Asset Management
- Asset Inventory
- Risk Classification
- Information Classification
- Business Continuity Planning
- Disaster Recovery Planning
- Documentation
- Google Sheets
- Cybersecurity Governance

## Technologies

- Microsoft Excel
- Google Sheets

## Project Outcome

A centralized asset inventory suitable for internal IT governance, cybersecurity audits, and ISO 27001-style asset management practices.
